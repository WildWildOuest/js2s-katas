# Christmas Tree

Given a number of rows render a tree with that many rows. For example, given 3 rows render the following tree:

```
  *
 ***
*****
  |
```

4 rows :

```
   *
  ***
 *****
*******
   |
```

Given 5 rows it should render the following:

```
    *
   ***
  *****
 *******
*********
    |
```
